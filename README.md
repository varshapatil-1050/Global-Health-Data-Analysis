#  Global Health Data Analysis

 Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

 Project Overview
This project analyzes global health indicators to identify trends and patterns affecting population health across countries. The analysis focuses on key factors such as life expectancy, GDP, healthcare expenditure, and mortality rates.

 Key Tasks Performed
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization using Matplotlib and Seaborn
- Correlation Analysis between health indicators
- Applied Random Forest Regression for prediction

Key Insights
- Higher GDP and healthcare spending are positively correlated with life expectancy  
- Countries with lower healthcare access show higher mortality rates  
- Strong relationships identified between socio-economic factors and health outcomes  

 Outcome
Developed a deeper understanding of real-world data analysis, feature relationships, and predictive modeling techniques using Python.

 Files Included
- global_health_indicator.ipynb → Main analysis notebook  
- global_health.csv → Dataset
