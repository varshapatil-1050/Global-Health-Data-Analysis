#  Global Health Data Analysis

# Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

# Project Overview
Analyzed global health data to identify key factors affecting life expectancy and mortality across countries.

# Key Tasks
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Analysis between GDP, healthcare, and life expectancy
- Built Random Forest Regression model for prediction

# Key Insights
- Higher GDP strongly correlates with increased life expectancy  
- Healthcare investment significantly impacts mortality rates  
- Socio-economic factors play a major role in health outcomes  

# Business Impact
This analysis helps in understanding which factors governments and organizations should prioritize to improve public health outcomes.
